## Search Anime

<Note>
  This endpoint does not require authentication.
</Note>

Search for anime by title or keywords.

<CodeGroup>

```bash cURL
curl -X GET "https://api.trackgeek.com/anime/search?query=naruto" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.trackgeek.com/anime/search?query=naruto', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.trackgeek.com/anime/search',
    params={'query': 'naruto'},
    headers={'Content-Type': 'application/json'}
)

data = response.json()
```

</CodeGroup>

### Query Parameters

<ParamField query="query" type="string" required>
  Search query string. Must be at least 3 characters long.
</ParamField>

      MyAnimeList ID
    </ResponseField>
    
    <ResponseField name="title" type="string">
      Anime title
    </ResponseField>
    
    <ResponseField name="imageUrl" type="string">
      Cover image URL
    </ResponseField>
    
    <ResponseField name="type" type="string">
      Anime type (TV, Movie, OVA, etc.)
    </ResponseField>
    
    <ResponseField name="numberOfEpisodes" type="number">
      Total number of episodes
    </ResponseField>
    
    <ResponseField name="status" type="string">
      Current airing status
    </ResponseField>
    
    <ResponseField name="synopsis" type="string">
      Synopsis/description
    </ResponseField>
    
    <ResponseField name="genres" type="array">
      Array of genre objects
    </ResponseField>
    
    <ResponseField name="studios" type="array">
      Array of studio objects
    </ResponseField>
    
    <ResponseField name="year" type="number">
      Release year
    </ResponseField>
  </Expandable>
</ResponseField>

## Get Anime Details

<Note>
  This endpoint does not require authentication.
</Note>

Retrieve detailed information about a specific anime by its ID.

<CodeGroup>

```bash cURL
curl -X GET "https://api.trackgeek.com/anime/details/123" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const animeId = 123;
const response = await fetch(`https://api.trackgeek.com/anime/details/${animeId}`, {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
```

```python Python
import requests

anime_id = 123
response = requests.get(
    f'https://api.trackgeek.com/anime/details/{anime_id}',
    headers={'Content-Type': 'application/json'}
)

data = response.json()
```

</CodeGroup>

## Refresh Anime

<Note>
  This endpoint requires authentication. Include your API token in the Authorization header.
</Note>

Refresh anime data from MyAnimeList to get the latest information.

<CodeGroup>

```bash cURL
curl -X POST "https://api.trackgeek.com/anime/refresh" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "id": 1735
  }'
```

```javascript JavaScript
const response = await fetch('https://api.trackgeek.com/anime/refresh', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    id: 1735
  })
});
```

```python Python
import requests

response = requests.post(
    'https://api.trackgeek.com/anime/refresh',
    headers={
        'Authorization': 'Bearer YOUR_TOKEN',
        'Content-Type': 'application/json'
    },
    json={'id': 1735}
)
```

</CodeGroup>

### Request Body

<ParamField body="id" type="integer" required>
  The MyAnimeList ID of the anime to refresh. Must be a valid integer.
</ParamField>

### Response

Returns HTTP 200 on successful refresh with no body content.

<ResponseExample>

```json Success Response
{}
```

</ResponseExample>
