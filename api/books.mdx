## Search Books

<Note>
  This endpoint does not require authentication.
</Note>

Search for books by title, author, or keywords.

<CodeGroup>

```bash cURL
curl -X GET "https://api.trackgeek.com/book/search?query=dune" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.trackgeek.com/book/search?query=dune', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.trackgeek.com/book/search',
    params={'query': 'dune'},
    headers={'Content-Type': 'application/json'}
)

data = response.json()
```

</CodeGroup>

### Query Parameters

<ParamField query="query" type="string" required>
  Search query string. Must be at least 3 characters long.
</ParamField>

    <ResponseField name="hardcoverId" type="number">
      Hardcover API ID
    </ResponseField>
    
    <ResponseField name="title" type="string">
      Book title
    </ResponseField>
    
    <ResponseField name="subtitle" type="string">
      Book subtitle
    </ResponseField>
    
    <ResponseField name="slug" type="string">
      URL-friendly slug
    </ResponseField>
    
    <ResponseField name="imageUrl" type="object">
      Cover image URLs in various sizes
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Book description/synopsis
    </ResponseField>
    
    <ResponseField name="releaseDate" type="string">
      Publication date (ISO 8601)
    </ResponseField>
    
    <ResponseField name="releaseYear" type="number">
      Publication year
    </ResponseField>
    
    <ResponseField name="numberOfPages" type="number">
      Total number of pages
    </ResponseField>
    
    <ResponseField name="literaryTypeId" type="number">
      Literary type identifier
    </ResponseField>
  </Expandable>
</ResponseField>

## Get Book Details

<Note>
  This endpoint does not require authentication.
</Note>

Retrieve detailed information about a specific book by its ID.

<CodeGroup>

```bash cURL
curl -X GET "https://api.trackgeek.com/book/details/123456" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const bookId = 123456;
const response = await fetch(`https://api.trackgeek.com/book/details/${bookId}`, {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
```

```python Python
import requests

book_id = 123456
response = requests.get(
    f'https://api.trackgeek.com/book/details/{book_id}',
    headers={'Content-Type': 'application/json'}
)

data = response.json()
```

</CodeGroup>

## Refresh Book

<Note>
  This endpoint requires authentication. Include your API token in the Authorization header.
</Note>

Refresh book data from Hardcover to get the latest information.

<CodeGroup>

```bash cURL
curl -X POST "https://api.trackgeek.com/book/refresh" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "id": 123456
  }'
```

```javascript JavaScript
const response = await fetch('https://api.trackgeek.com/book/refresh', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    id: 123456
  })
});
```

```python Python
import requests

response = requests.post(
    'https://api.trackgeek.com/book/refresh',
    headers={
        'Authorization': 'Bearer YOUR_TOKEN',
        'Content-Type': 'application/json'
    },
    json={'id': 123456}
)
```

</CodeGroup>

### Request Body

<ParamField body="id" type="integer" required>
  The Hardcover ID of the book to refresh. Must be a valid integer.
</ParamField>

### Response

Returns HTTP 200 on successful refresh with no body content.

<ResponseExample>

```json Success Response
{}
```

</ResponseExample>
