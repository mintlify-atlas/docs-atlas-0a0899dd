## Search Manga

<Note>
  This endpoint does not require authentication.
</Note>

Search for manga by title or keywords.

<CodeGroup>

```bash cURL
curl -X GET "https://api.trackgeek.com/manga/search?query=one%20piece" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.trackgeek.com/manga/search?query=one%20piece', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.trackgeek.com/manga/search',
    params={'query': 'one piece'},
    headers={'Content-Type': 'application/json'}
)

data = response.json()
```

</CodeGroup>

### Query Parameters

<ParamField query="query" type="string" required>
  Search query string. Must be at least 3 characters long.
</ParamField>

<ResponseField name="mangas" type="array">

    <ResponseField name="malId" type="number">
      MyAnimeList ID
    </ResponseField>
    
    <ResponseField name="title" type="string">
      Manga title
    </ResponseField>
    
    <ResponseField name="imageUrl" type="string">
      Cover image URL
    </ResponseField>
    
    <ResponseField name="type" type="string">
      Manga type (Manga, Novel, Manhwa, Manhua, etc.)
    </ResponseField>
    
    <ResponseField name="numberOfChapters" type="number">
      Total number of chapters
    </ResponseField>
    
    <ResponseField name="numberOfVolumes" type="number">
      Total number of volumes
    </ResponseField>
    
    <ResponseField name="status" type="string">
      Publishing status
    </ResponseField>
    
    <ResponseField name="synopsis" type="string">
      Synopsis/description
    </ResponseField>
    
    <ResponseField name="genres" type="array">
      Array of genre objects
    </ResponseField>
    
    <ResponseField name="authors" type="array">
      Array of author objects
    </ResponseField>
  </Expandable>
</ResponseField>

## Get Manga Details

<Note>
  This endpoint does not require authentication.
</Note>

Retrieve detailed information about a specific manga by its ID.

<CodeGroup>

```bash cURL
curl -X GET "https://api.trackgeek.com/manga/details/13" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const mangaId = 13;
const response = await fetch(`https://api.trackgeek.com/manga/details/${mangaId}`, {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
```

```python Python
import requests

manga_id = 13
response = requests.get(
    f'https://api.trackgeek.com/manga/details/{manga_id}',
    headers={'Content-Type': 'application/json'}
)

data = response.json()
```

</CodeGroup>

## Refresh Manga

<Note>
  This endpoint requires authentication. Include your API token in the Authorization header.
</Note>

Refresh manga data from MyAnimeList to get the latest information.

<CodeGroup>

```bash cURL
curl -X POST "https://api.trackgeek.com/manga/refresh" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "id": 13
  }'
```

```javascript JavaScript
const response = await fetch('https://api.trackgeek.com/manga/refresh', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    id: 13
  })
});
```

```python Python
import requests

response = requests.post(
    'https://api.trackgeek.com/manga/refresh',
    headers={
        'Authorization': 'Bearer YOUR_TOKEN',
        'Content-Type': 'application/json'
    },
    json={'id': 13}
)
```

</CodeGroup>

### Request Body

<ParamField body="id" type="integer" required>
  The MyAnimeList ID of the manga to refresh. Must be a valid integer.
</ParamField>

### Response

Returns HTTP 200 on successful refresh with no body content.

<ResponseExample>

```json Success Response
{}
```

</ResponseExample>
