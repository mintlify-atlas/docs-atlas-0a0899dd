## Search Movies

<Note>
  This endpoint does not require authentication.
</Note>

Search for movies by title or keywords.

<CodeGroup>

```bash cURL
curl -X GET "https://api.trackgeek.com/movie/search?query=inception" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.trackgeek.com/movie/search?query=inception', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.trackgeek.com/movie/search',
    params={'query': 'inception'},
    headers={'Content-Type': 'application/json'}
)

data = response.json()
```

</CodeGroup>

<ParamField query="query" type="string" required>
  Search query string. Must be at least 3 characters long.
</ParamField>

## Get Movie Details

<Note>
  This endpoint does not require authentication.
</Note>

Retrieve detailed information about a specific movie by its ID.

<CodeGroup>

```bash cURL
curl -X GET "https://api.trackgeek.com/movie/details/27205" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const movieId = 27205;
const response = await fetch(`https://api.trackgeek.com/movie/details/${movieId}`, {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
```

```python Python
import requests

movie_id = 27205
response = requests.get(
    f'https://api.trackgeek.com/movie/details/{movie_id}',
    headers={'Content-Type': 'application/json'}
)

data = response.json()
```

</CodeGroup>

## Refresh Movie

<Note>
  This endpoint requires authentication. Include your API token in the Authorization header.
</Note>

Refresh movie data from TMDB to get the latest information.

<CodeGroup>

```bash cURL
curl -X POST "https://api.trackgeek.com/movie/refresh" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "id": 27205
  }'
```

```javascript JavaScript
const response = await fetch('https://api.trackgeek.com/movie/refresh', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    id: 27205
  })
});
```

```python Python
import requests

response = requests.post(
    'https://api.trackgeek.com/movie/refresh',
    headers={
        'Authorization': 'Bearer YOUR_TOKEN',
        'Content-Type': 'application/json'
    },
    json={'id': 27205}
)
```

</CodeGroup>

### Request Body

<ParamField body="id" type="integer" required>
  The TMDB ID of the movie to refresh. Must be a valid integer.
</ParamField>

### Response

Returns HTTP 200 on successful refresh with no body content.

<ResponseExample>

```json Success Response
{}
```

</ResponseExample>
