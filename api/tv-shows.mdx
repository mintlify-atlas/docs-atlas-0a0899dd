## Search TV Shows

<Note>
  This endpoint does not require authentication.
</Note>

Search for TV shows by title or keywords.

<CodeGroup>

```bash cURL
curl -X GET "https://api.trackgeek.com/tv/search?query=breaking%20bad" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.trackgeek.com/tv/search?query=breaking%20bad', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.trackgeek.com/tv/search',
    params={'query': 'breaking bad'},
    headers={'Content-Type': 'application/json'}
)

data = response.json()
```

</CodeGroup>

<ParamField query="query" type="string" required>
  Search query string. Must be at least 3 characters long.
</ParamField>

## Get TV Show Details

<Note>
  This endpoint does not require authentication.
</Note>

Retrieve detailed information about a specific TV show by its ID.

<CodeGroup>

```bash cURL
curl -X GET "https://api.trackgeek.com/tv/details/1396" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const tvShowId = 1396;
const response = await fetch(`https://api.trackgeek.com/tv/details/${tvShowId}`, {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
```

```python Python
import requests

tv_show_id = 1396
response = requests.get(
    f'https://api.trackgeek.com/tv/details/{tv_show_id}',
    headers={'Content-Type': 'application/json'}
)

data = response.json()
```

</CodeGroup>

## Refresh TV Show

<Note>
  This endpoint requires authentication. Include your API token in the Authorization header.
</Note>

Refresh TV show data from TMDB to get the latest information.

<CodeGroup>

```bash cURL
curl -X POST "https://api.trackgeek.com/tv/refresh" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "id": 1396
  }'
```

```javascript JavaScript
const response = await fetch('https://api.trackgeek.com/tv/refresh', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    id: 1396
  })
});
```

```python Python
import requests

response = requests.post(
    'https://api.trackgeek.com/tv/refresh',
    headers={
        'Authorization': 'Bearer YOUR_TOKEN',
        'Content-Type': 'application/json'
    },
    json={'id': 1396}
)
```

</CodeGroup>

### Request Body

<ParamField body="id" type="integer" required>
  The TMDB ID of the TV show to refresh. Must be a valid integer.
</ParamField>

### Response

Returns HTTP 200 on successful refresh with no body content.

<ResponseExample>

```json Success Response
{}
```

</ResponseExample>
